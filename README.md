# Expo Cloud Starter Template

A minimal Expo + EAS starter template for building Android and web apps
without requiring a local PC or Android Studio.

## Features
- Expo SDK
- Expo Router (file-based navigation)
- TypeScript-ready
- EAS cloud builds
- Android APK support
- Works with GitHub Codespaces
- Web + Mobile compatible

## Use cases
- Market & commerce apps
- NGO & community tools
- Internal business apps
- Data collection apps
- Offline-first apps
- Rapid MVPs

## Getting Started

1. Clone or copy this repository
2. Update `app.json`
   - `name`
   - `slug`
   - icons (optional)
3. Install dependencies:
   ```sh
   npm install
